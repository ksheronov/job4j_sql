# job4j_sql

Репозиторий для практических заданий по курсу SQL в рамках обучения на платформе job4j.

## Описание

В этом репозитории хранятся SQL-скрипты для:
- Создания таблиц
- Модификации структуры БД (ALTER)
- Наполнения тестовыми данными
- Выполнения запросов

Каждая задача имеет свою директорию с понятным именем и номером.
